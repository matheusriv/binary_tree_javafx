# Binary Tree JavaFX

Exercise of the PROGRAMMING LANGUAGE II 2021.2 course at UFRN.

A way to visualize binary search tree, balanced binary search tree and red black tree.

Using JDK 11.0.13 and JafaFX 17.0.2
